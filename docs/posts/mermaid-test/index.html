Mermaid 图表测试

Saturday, Oct 4, 2025 | 1 minute read | Updated at Saturday, Oct 4, 2025

@ fancivez

Dream 主题原生支持 Mermaid 图表！下面是一些示例。

流程图示例

graph TD
    A[开始] -->|初始化| B{检查条件}
    B -->|条件1| C[执行操作1]
    B -->|条件2| D[执行操作2]
    C --> E[结束]
    D --> E

时序图示例

sequenceDiagram
    participant U as 用户
    participant A as API网关
    participant S as 服务端
    participant DB as 数据库

    U->>A: 发送请求
    A->>S: 转发请求
    S->>DB: 查询数据
    DB-->>S: 返回结果
    S-->>A: 处理响应
    A-->>U: 返回数据

架构图示例

graph LR
    A[监控系统] -->|告警| B[告警聚合器]
    B -->|结构化告警| C[LLM诊断引擎]
    D[日志系统] -->|相关日志| C
    E[配置管理] -->|系统配置| C
    C -->|诊断结果| F[自动修复系统]
    C -->|诊断报告| G[运维仪表板]

甘特图示例

gantt
    title 项目开发计划
    dateFormat YYYY-MM-DD
    section 需求分析
    需求调研           :a1, 2025-01-01, 7d
    需求文档           :a2, after a1, 5d
    section 开发阶段
    后端开发           :b1, 2025-01-13, 14d
    前端开发           :b2, 2025-01-13, 14d
    section 测试上线
    测试               :c1, after b1, 7d
    上线               :c2, after c1, 2d
