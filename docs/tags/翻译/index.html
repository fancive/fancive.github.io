go string to byte

0x01 背景

最近在review其他同学代码发现将 string转[]byte的时候，得到的[]byte的容量是会变的, 比如如下一段代码的输出是32

func main() {
	s1 := "abc"
	fmt.Println(cap([]byte(s1)))
}

0x02 探究

-N 禁止优化 -S 输出汇编代码 -l 禁止内联
