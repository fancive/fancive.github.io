runtime.assertI2I

代码位置是 src/runtime/iface.go, 本函数断言 一个iface(非空接口)可以转为另外一个 iface，如果失败直接panic

iface

在说 runtime.getitab()之前有必要说明下 iface的数据结构， 它由两部分组成

  1. itab 类型信息
    1. Itab.inter 可以理解为 interface type, 这是一种抽象表示
    2. Itab._type 可以理解为 concrete type, 这是一个实体类型的表示
  2. data 数据信息

_type

runtime.getitab

  1. 因为itab由两部分组成, 接口类型 interfaceType 和实体类型 _type, 因此getitab的参数也是这两部分
  2. 这里会从全局的 hashtable 中寻找2次itab ,如果没有就会初始化一个新的
  3. 两次查找中，第一次无锁，第二次有锁，第二次加锁是为了保证不会因为竞争导致初始化多个itab
