go-goland使用

发表于 2020-08-09 | 分类于 编程语言

最近看了一下 TalkGo 第50期 第 50 期 2019-06-27 GoLand Tips & Tricks， 发现有一些冷门但是很好用的功能

[toc]

全局快捷键

可以使用 find action 每次查询并慢慢积累自己记忆的快捷键

  1. command+shift+a / find action (用于快速查询快捷键)
  2. Shift + shift / 全局搜索(所有文件、函数、变量和action)

错误处理技巧

对于错误处理我之前只会在报错的代码上使用 (command+enter)

next hightlight error

我们通常在ide中最右侧的minimap中看到当前文件的语法错误标识，但是往往需要鼠标点击才能定位到错误位置，这时候借助(next hightlight error)可以实现快速跳转, 类似vscode中的(shift+option+n)

重构

对于重构我之前只知道

refactor this (ctrl+t)

可以将选中的代码块重构为变量、独立函数(同时迁移到其他package)

打开最近的文件

之前一般借助 command+b来跳转到之前修改的文件

  1. command+e 打开最近文件()
  2. command+shift+e 打开最近位置(recent position)

playground

command+shift+n / New scratch file 新建一个单独的 playground，可以用来验证想法、做早期原型
