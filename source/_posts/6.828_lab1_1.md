清空段寄存器 DS、ES、SS

```s
  # Zero data segment registers DS, ES, and SS.

  xorw    %ax,%ax             # Set %ax to zero
  movw    %ax,%ds             # -> Data Segment
  movw    %ax,%es             # -> Extra Segment
  movw    %ax,%ss             # -> Stack Segment
```
> 确认 0x64和0x60 输出端口的高位为2，使第21位地址正常工作

```assemble
seta20.1:
  inb     $0x64,%al               # in byte 从端口 0x64 读 1 bype到 %al  0x60端口是数据端口,可以读出键盘数据,0x64是控制端口,用来发出控制信号
  testb   $0x2,%al                # 逻辑与运算结果为零,就把ZF(零标志)置1;
  jnz     seta20.1                   # jump if not zero (结果不为0则设置ZF为1,跳转)

  movb    $0xd1,%al               # 
  outb    %al,$0x64                 # 写1 Byte 到 0x64端口
  
seta20.2:
  inb     $0x64,%al               # Wait for not busy
  testb   $0x2,%al
  jnz     seta20.2

  movb    $0xdf,%al               # 0xdf -> port 0x60
  outb    %al,$0x60  
```

引导加载器执行 lgdt指令来把指向 gdt 的指针 gdtdesc加载到全局描述符表（GDT）寄存器中。将 %cr0 中的 CR0_PE 位置为1，从而开启保护模式

```c
  lgdt    gdtdesc
  movl    %cr0, %eax
  orl     $CR0_PE, %eax
  movl    %eax, %cr0

```
